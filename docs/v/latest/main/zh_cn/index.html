                                                                          法术

                                                                          能以魔法影响到世界的图案和操作。

                                                                          标记法术

                                                                          设置标记 (map, marker →)

                                                                          • 渲染方式
                                                                            • 配色
                                                                              浏览器不支持图案的可视化。图案代码见此：aawwaawqaawwaaw

                                                                              将所给标记以其标识加至地图。

                                                                              会在此时判定是否在影响范围内。


                                                                              各类标记的判定方式不一。具体如下：

                                                                              圆形标记判定其圆心位置及由此沿各轴向偏移半径长度的各位置。

                                                                              矩形标记判定其两角。

                                                                              多边形和折线标记判定所有点。

                                                                              图标标记判定其位置。


                                                                              移除标记 (map, str →)

                                                                              • 渲染方式
                                                                                • 配色
                                                                                  浏览器不支持图案的可视化。图案代码见此：wddweddwwddwedd

                                                                                  移除地图中使用所给标识的标记。
